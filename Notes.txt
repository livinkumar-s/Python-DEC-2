Python:
------------
Python is a high-level, interpreted, general-purpose programming language.
       
            Compilation            Interpretation
Human Written    --->     Byte Code  --->     Machine Code
Code                (000 1111 111000)    (Computer can Understand)


Extension of python ---> .py
command to execute python file ---> python filename.py

Print() Function
-------------------
print(data) --> this data will be shown at the output

input() Function 
-----------------
input() ---> to prompt data from user.
always returns string value

Syntax(Set of rules)
---------------------
>> It is a case-sensitive programming language.(a not equal to A).
>> Indentation matters. (whitespace before a line of code)

Comments 
---------
>> To increase readability.
To create a single line comment --> #
To create multiple line comments --> '''   '''

Variable
---------
>> A variable is a named storage to store a data.
varName=data

Naming Rules 
--------------
>> Should contain only letters, numerics and -,_
>> Should start with only letters
>> No reserved keywords

Datatypes 
----------
Integer --> Whole Numbers 
Float --> Decimal Numbers 
String ---> Sequence of chars' enclosed in either '' or "". Ex: "Hello"
Concatination: str1+str2
Formatted String 

Boolean ---> True/False

Type Casting
-------------
type(obj) ---> finds the datatype.
>>Converting a data from one datatype to another
int(obj) --> converts into int 
str(obj)
float(obj)
bool(obj)
